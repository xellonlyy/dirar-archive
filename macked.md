# ✦ MacKed Guide

Easily download and patch the latest version of any Creative Cloud software on **MacOS** using MacKed’s Adobe Activation Tool, an unlimited selection of apps, faster updates, and other benefits.

::: danger
Mixing MacKed with other methods on your machine is **not recommended**, as it will inevitably lead to issues. Run Adobe's Creative Cloud Cleaner Tool to completely remove previous Adobe apps if this warning applies to you.
:::

## Online Method

1. Download Adobe Creative Cloud, you must create an account and login — feel free to use a temporary email. Do NOT install Adobe Genuine Service (AGS) if prompted. Close the application once installed.

**Creative Cloud:** [https://helpx.adobe.com/download-install/apps/download-install-apps/creative-cloud-apps/download-creative-cloud-desktop-app-from-web.html](https://helpx.adobe.com/download-install/apps/download-install-apps/creative-cloud-apps/download-creative-cloud-desktop-app-from-web)

1. Download “Adobe Activation Tool 1.2.4 [MacKed].dmg” below. Drag the application to the blue folder to install the application as shown in the picture.

**Adobe Activation Tool 1.2.5:** [**Download Link**](https://www.mediafire.com/file_premium/8y30axdl18ourba/Adobe_Activation_Online_Tool_1.2.5.zip/file)

> On the off chance this guide falls behind an update, you can find the latest version on their official website [https://macked.app](https://macked.app) — login required. Compatibility and restrictions can also be found on their website.

1. After installation, go to Settings > Privacy & Security > App Management. Click the plus symbol, and add Adobe Activation tool which can be selected from your applications folder. Ensure the switch is turned on. **This is important.**
2. Open Adobe Activation Tool, click the plus icon and select ` Applications/Utilities/Adobe Creative Cloud/ACC/Creative Cloud.app` — after, click “start activation”.
3. Open Creative Cloud and install your desired software (After Effects, Premiere Pro, etc). You should see an “install” button as opposed to “free trial” next to apps. Restart your device if necessary and close all Adobe apps once installed.
4. Open Adobe Activation Tool, click the plus symbol, and add your desired application (example: `Applications/Adobe After Effects 2025/Adobe After Effects 2025.app`) and select “start activation”.

The application will now be patched, enjoy.

> Open Creative cloud > Go to Preferences and in the "General" tab disable the following settings: Launch Creative Cloud at login; Run Creative Cloud in the background after closing; Automatically keep Creative Cloud up to date.
