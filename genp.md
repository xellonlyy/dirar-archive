# ✦ GenP Guide

Easily download and patch the latest version of any Creative Cloud software on **Windows** using GenP, an unlimited selection of apps, faster updates, and other benefits.

::: danger
Mixing GenP with other methods on your machine is **not recommended**, as it will inevitably lead to issues. Run Adobe's Creative Cloud Cleaner Tool to completely remove previous Adobe apps if this warning applies to you.
:::

**Adobe Creative Cloud Cleaner Tool:** [https://swupmf.adobe.com/webfeed/CleanerTool/win/AdobeCreativeCloudCleanerTool.exe](https://swupmf.adobe.com/webfeed/CleanerTool/win/AdobeCreativeCloudCleanerTool.exe)

## 

1. Download Adobe Creative Cloud. You must create an account and sign in — Feel free to use a temp email. Do NOT install Adobe Genuine Service (AGS) if prompted.

**Creative Cloud:** [https://helpx.adobe.com/download-install/apps/download-install-apps/creative-cloud-apps/download-creative-cloud-desktop-app-from-web.html](https://helpx.adobe.com/download-install/apps/download-install-apps/creative-cloud-apps/download-creative-cloud-desktop-app-from-web)

1. Go to Menu > File > Preferences and in the "General" tab disable the following settings: Launch Creative Cloud at login; Run Creative Cloud in the background after closing; Automatically keep Creative Cloud up to date.
2. Close Creative Cloud once finished. (Menu > File > Quit Creative Cloud)
3. Download GenP from the official GenP website.

**Official GenP link:** [https://gen.paramore.su](https://gen.paramore.su)

The Binary download is recommended for beginners as it’s ready to use, while source is intended for more advanced users.

> Windows Defender, along with certain third-party programs, are known to wrongfully block GenP. You must whitelist or exclude the file. You may also disable real-time protection completely, but do not forget to turn it back on afterwards.

1. Right-click the zip > “Extract all” (GenP.vX.X.X-CGP.zip) and run GenP (GenP-vX.X.X.exe) as administrator.
2. In the main tab, click "Search". You should be prompted three files underneath Creative Cloud, ensure that all three are selected and click “Patch”.
3. Open Creative Cloud and install your desired applications (Premiere, After Effects, and so on) in the Apps menu. Once the applications are installed, close Creative Cloud and all respective Adobe apps.
4. Run GenP (GenP-vX.X.X.exe) as administrator and click “search”. This time, select the files for your program(s) (Premiere, After Effects, etc) and click “patch”.

The application will now be patched, enjoy.
